Desarrollo de un sitio web tipo catálogo para la marca Mountain Life, enfocado en la presentación clara y atractiva de filtros de agua. Se implementó una estructura optimizada para facilitar la navegación del usuario y mejorar la conversión.

El proyecto incluye gestión de productos mediante base de datos, diseño responsivo y optimización SEO para mejorar el posicionamiento en buscadores. Se priorizó la velocidad de carga y una experiencia de usuario intuitiva que permita encontrar información de forma rápida y efectiva.
